# COMMAND

**The AI-powered operating system for your household.**

COMMAND brings order, foresight, and automation to everything a family is accountable for — insurance, legal, finances, home maintenance, healthcare, and more.

## Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

### Deployment to Vercel

1. Push this repository to GitHub
2. Import the repository in [Vercel](https://vercel.com)
3. Vercel will auto-detect the Vite configuration
4. Deploy!

## Tech Stack

- **React 18** - UI framework
- **TypeScript** - Type safety
- **Vite** - Build tool & dev server
- **Tailwind CSS** - Styling
- **Lucide React** - Icons
